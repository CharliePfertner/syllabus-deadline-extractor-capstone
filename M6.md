## Github Copilot
I noticed that Github Copilot can assist you while you are typing your code, while Claude does not.

## Cursor

I would prefer Cursor for large, complex development where visual multi-file navigation, interactive UI/UX tweaking, and localized code refactoring are key, whereas Claude Code shines in terminal-centric workflows, rapid test-driven iteration loops, and executing multi-step repository-wide tasks directly from the command line.

## Gemini

I feel as though Claude was more accurate, as it describes the students' objective with the sandbox in greater detail than Gemini.

## Decision matrix

| Situation | Tool I'd Reach For | Why |
| Quickly creates a python script | Claude.ai (Web App) | For high-level brainstorming and doesn't clutter workspace |
| Writing standard boilerplate code | Github Copilot | For assisting you while you code |
| Makes deep multi-file edits while looking at the code visually |Cursor | Perfect for when you need to see changes side-by-side |
| Answer questions about recently released API changes | Gemini CLI | Built-in Google search grounding and a massive 1-million+ token context window |
| Finding deep bugs and self-correcting until it fixes these said bugs | Claude Code | Operates entirely in a loop and runs terminal commands |
